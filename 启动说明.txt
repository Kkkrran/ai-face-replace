AI换脸工具 - 快速启动指南
================================

1. 安装依赖
   pip install -r requirements.txt

2. 配置API密钥（可选）
   编辑 config.py 文件，或设置环境变量：
   set ARK_API_KEY=your_api_key_here

3. 启动应用
   python app.py

4. 访问应用
   打开浏览器访问：http://localhost:5000

5. 使用说明
   - 上传衣物图片和人脸图片
   - 输入提示词（可选）
   - 点击"生成图片"按钮
   - 等待生成完成，查看结果

注意事项：
- 确保网络连接正常
- API调用可能需要较长时间（最多2分钟）
- 图片格式支持：JPG、PNG、WEBP、BMP
- 建议图片大小不超过10MB
